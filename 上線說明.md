# Menu English Studio 上線說明

這個資料夾是一個靜態網站。上線時只需要上傳整個 `web` 資料夾內容，不需要上傳 PDF 或 MP4。

## 包含內容

- `index.html`：網站頁面
- `styles.css`：視覺樣式
- `app.js`：互動、進度、發音、筆記
- `data.js`：課程、單字、菜單例句、句型資料
- `assets/`：網站使用的圖片素材

## 可上線的平台

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
- 一般網站空間

## 注意事項

發音功能使用瀏覽器內建語音。不同手機或電腦的聲音會略有差異。
